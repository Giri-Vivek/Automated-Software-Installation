This is use for installing Automatically software listed in packages.txt
Updates package lists before installation.
Logs all installation results to install_log.txt
Skips empty lines safely.
Shows real-time installation progress.
Simple and easy to customize.

Automated-Software-Installation/
│
├── install.sh           # Main installation script
├── packages.txt         # List of packages to install
└── install_log.txt      # Log file generated after running the script
